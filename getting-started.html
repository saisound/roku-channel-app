Step 1: Create Your First Tile

Start with a simple 16:9 medium tile - the most common format for video content.

// Create a basic video tile const myFirstTile = TileComponent.createTile('16:9', 'medium', { title: 'The Mandalorian', subtitle: 'Sci-Fi â€¢ Action' }); // Add it to a container document.getElementById('step1-result').appendChild(myFirstTile);

Step 2: Try Different Sizes

See how the same content looks in small, medium, and large sizes.

// Create tiles in different sizes const sizes = ['small', 'medium', 'large']; const container = document.getElementById('step2-result'); sizes.forEach(size => { const tile = TileComponent.createTile('16:9', size, { title: `${size.charAt(0).toUpperCase() + size.slice(1)} Tile`, subtitle: 'Same content, different size' }); container.appendChild(tile); });

Step 3: Explore Aspect Ratios

Different content types work better with different aspect ratios.

// Create tiles with different aspect ratios const ratios = [ { type: 'promo', size: 'large', title: 'Promo Banner' }, { type: '21:9', size: 'medium', title: 'Cinematic' }, { type: '16:9', size: 'medium', title: 'Standard Video' }, { type: '4:3', size: 'small', title: 'Classic TV' }, { type: '2:3', size: 'medium', title: 'Movie Poster' } ]; const container = document.getElementById('step3-result'); ratios.forEach(config => { const tile = TileComponent.createTile(config.type, config.size, { title: config.title, subtitle: `${config.type} aspect ratio` }); container.appendChild(tile); });

Step 4: Add Continue Watching Progress

Add progress bars to show viewing progress on content.

// Create continue watching tiles with progress const shows = [ { title: 'Breaking Bad', progress: 45 }, { title: 'Stranger Things', progress: 78 }, { title: 'The Crown', progress: 23 } ]; const container = document.getElementById('step4-result'); shows.forEach(show => { const tile = TileComponent.createTile('16:9', 'medium', { title: show.title, subtitle: 'Continue Watching' }, { continueWatching: true, progress: show.progress }); container.appendChild(tile); });

Step 5: Create a Complete Content Row

Build a complete row with title and multiple tiles - ready to add to your app.

// Create a complete content row const rowConfig = [ { type: '16:9', size: 'large', content: { title: 'Featured Movie', subtitle: 'Action â€¢ 2024' } }, { type: '16:9', size: 'medium', content: { title: 'Popular Series', subtitle: 'Drama â€¢ Comedy' } }, { type: '2:3', size: 'small', content: { title: 'New Release', subtitle: 'Thriller â€¢ 2024' } }, { type: '16:9', size: 'medium', content: { title: 'Trending Now', subtitle: 'Sci-Fi â€¢ Adventure' } } ]; const completeRow = TileComponent.createTileRow('Featured Content', rowConfig); document.getElementById('step5-result').appendChild(completeRow);
