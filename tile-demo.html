Roku Channel Tile Component System

This demo showcases the flexible tile component system that supports different aspect ratios and sizes based on your specifications. Each tile type maintains proper proportions and responsive behavior.

Promo Banners

Wide promotional banners for featured content and announcements.

Small: 1716×256 (scaled)
Large: 1596×582 (scaled)
// Create promo tiles const promoSmall = TileComponent.createTile('promo', 'small', { title: 'Summer Movie Festival', subtitle: 'Watch now for free' }); const promoLarge = TileComponent.createTile('promo', 'large', { title: 'Exclusive Premiere Weekend', subtitle: 'New releases every Friday' });

21:9 Cinematic Banners

Ultra-wide banners perfect for cinematic content and hero images.

Small: 411×176
Medium: 556×238
Large: 846×363
// Create 21:9 banner tiles const banners = ['small', 'medium', 'large'].map(size => TileComponent.createTile('21:9', size, { title: `Epic Movie ${size.charAt(0).toUpperCase() + size.slice(1)}`, subtitle: 'Action • Adventure' }) );

16:9 Standard Video Tiles

Standard widescreen format for most video content.

Small: 411×231
Medium: 556×312
Large: 846×474
// Create 16:9 video tiles const videoTile = TileComponent.createTile('16:9', 'medium', { title: 'The Mandalorian', subtitle: 'Sci-Fi • Action' }, { continueWatching: true, progress: 65 });

4:3 Classic TV Format

Traditional TV format for classic shows and retro content.

Small: 324×243
// Create 4:3 classic TV tile const classicTile = TileComponent.createTile('4:3', 'small', { title: 'Classic Comedy Hour', subtitle: 'Comedy • Vintage' });

2:3 Portrait/Poster Format

Vertical format perfect for movie posters and portrait content.

// Create 2:3 poster tiles const posterTile = TileComponent.createTile('2:3', 'large', { title: 'Blockbuster Movie', subtitle: 'Now Streaming' });

Mixed Layout Example

Demonstration of how different tile types work together in a content row.

// Create a mixed layout row const mixedRowConfig = [ { type: 'promo', size: 'large', content: { title: 'Special Event' } }, { type: '16:9', size: 'medium', content: { title: 'Popular Show' } }, { type: '2:3', size: 'small', content: { title: 'New Movie' } }, { type: '21:9', size: 'medium', content: { title: 'Epic Series' } } ]; const mixedRow = TileComponent.createTileRow('Featured Content', mixedRowConfig);
